## About This Project

This is my personal portfolio website, designed to showcase my skills, projects, and contact information. I built it using HTML5 and CSS3, with a focus on mobile-first responsive design. The site demonstrates modern layout techniques like Flexbox and CSS Grid, uses CSS variables for colors and spacing, and follows accessibility best practices.  

---

## Sections

### Home / Hero
The home section introduces me with my name, a tagline, and a call-to-action button linking to the contact section. I used a linear gradient background and centered the content with Flexbox for a clean, modern look.

### About
This section features my profile picture, a short professional bio, and a list of my skills, including:  
HTML5, CSS3, JavaScript, Responsive Design, Flexbox, CSS Grid, and Git/GitHub.  
I used Flexbox to layout the image and text side by side on larger screens.

### Projects
I created three project cards, each with an image, title, description, and links:  
1. **Personal Portfolio Website** – A responsive site showcasing my work.  
2. **Product Landing Page** – Modern landing page with clean layout and typography.  
3. **Todo List Web App** – Simple productivity web application.  

The project cards are laid out using CSS Grid, which makes them adjust automatically to different screen sizes.

### Contact
A contact form allows visitors to send messages directly. It includes validation for name, email, subject, and message, ensuring the user inputs all required information.

### Footer
The footer includes copyright information and links to my social profiles (GitHub, LinkedIn, Twitter).

## Key Features & Techniques

- **CSS Variables:** For colors, spacing, and font sizes  
- **Flexbox:** Hero section, navigation, and About layout  
- **CSS Grid:** Project cards  
- **Responsive Design:** Mobile-first with breakpoints at 768px, 1024px, and 1200px  
- **Accessibility:** Alt text for images, proper labels on form inputs, clear heading hierarchy  
- **Clean Design:** Modern typography, consistent spacing, and color palette
